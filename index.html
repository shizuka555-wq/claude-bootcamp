よくあるご質問

Q1. 初心者でも大丈夫？

大丈夫です。
というより、初心者のほうが伸びます。

変な自己流が入ってないぶん、Claudeとの付き合い方を素直にインストールできます。

今回は「黒い画面アレルギー」の方も想定した設計にしています。

Q2. Claude Codeは無料でも使えますか？

Claude Code 自体のインストールは無料です。
ただし、実際に動かすには Anthropic のプラン契約または API 利用料が別途必要です。

最低限、Pro プラン（月額 $20〜）があれば始められます。
※ ブートキャンプ代には Claude のプラン料金は含まれません。

どこまで使い込むかによって最適なプランは変わるので、
「自分はどのプランがいい？」という相談もブートキャンプの中で受け付けます。

Q3. ClaudeCode触ったことなくてもいける？

大丈夫です。

初回セミナーで解説した内容を前提にスタートしますが、アーカイブ視聴も可能です。

ゼロからでも、約2ヶ月で十分に使える状態まで持っていけます。

Q4. ライブに参加できない回がある場合は？

すべてのライブはアーカイブ視聴が可能です。

質問はグループチャットにいつでも投稿OK。
作業Zoom会で個別にフォローします。

全回リアルタイム参加が前提の設計ではないので、安心してお申し込みください。

Q5. キャンセル可能？

お支払い後のキャンセル・返金は承っておりません。

Q6. 分身にやらせたい業務がまだ明確じゃなくても参加できる？

全く問題ありません。

むしろ「何をAIに任せればいいかわからない」状態からスタートする方がほとんどです。

先着5名のスタートアップセッションで、あなたの業務棚卸しから一緒にやります。

本編開講までに、あなた専用の「分身設計図」ができている状態を目指します。

Q7. Windows／Mac両方OK？

両方OKです。

Claude Code（デスクトップアプリ版）は、WindowsでもMacでも動きます。

※ 私はMacユーザーです。

Q8. どのくらいの時間投下が必要？

目安は週2〜3時間。

  • 週1ライブセミナー：60〜90分
  • 週1質問OK作業Zoom会：60〜90分
  • 自分で手を動かす時間：30〜60分

忙しい週はアーカイブ視聴で追いつけます。

Q9. ある程度ClaudeCode使ってる人も対象ですか？

対象です。

初心者から、すでに触っている方まで幅広く対応できる設計です。

ある程度使っている方には、

  • mdファイルを活用した業務設計
  • UTAGE・マイスピー等の外部ツール連携
  • 入金管理・投稿自動化など業務への具体的な組み込み

など、「実務レベルの活用」に進む内容もお届けします。

質問Zoom会では、あなたのレベルに合わせて個別に深掘りできます。
